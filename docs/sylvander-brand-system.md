# Sylvander Agent Brand System

> Status: Selected brand baseline
>
> Version: 2.0
>
> Date: 2026-07-12

## 1. Brand ambition

Sylvander is not a chat mascot or a terminal utility with an AI feature. It is an agent workspace that observes context, makes durable decisions, coordinates tools and subagents, moves work forward, and verifies outcomes.

The brand must feel:

- Intelligent without using a generic brain, spark, or eye symbol.
- Active without becoming an aggressive arrow or automation logo.
- Technical without collapsing into angle brackets or a command prompt cliché.
- Distinctive at app-icon scale and meaningful inside a monospace terminal.
- Calm and premium enough to sit beside frontier-model products without imitating them.

Claude Fable is a quality benchmark for ambitious, long-running agent work and high-fidelity execution, not a visual template to copy. Sylvander's expression is more operational: intelligence becomes directed action inside a durable workspace.

## 2. Selected concept: Convergence Core

The final mark has three semantic stages:

1. **Convergence** — multiple violet inputs represent context, tools, memory, and subagents.
2. **Core** — the coral diamond represents a deliberate decision with identity and memory.
3. **Vector** — the cyan forward stroke turns judgment into action.

The shape is balanced without being enclosed. It retains an abstract trace of crab energy through distributed side inputs, but it does not draw claws, eyes, legs, or a shell. It is a system symbol rather than a mascot.

## 4. Brand language in the UI

Brand is distributed semantically rather than repeated decoratively:

- **Signal Violet** `#9B8CFF`: Sylvander intelligence, agent identity, reasoning, main-agent core.
- **Intent Coral** `#FF806F`: user instruction, selection, input focus, explicit choice.
- **Action Cyan** `#6DD6E8`: live execution, streaming, forward progress.
- **Verified Teal** `#74D6B7`: evidence-backed completion.
- **Warm Ivory** `#F0ECE4`: primary readable content.

This creates a visible grammar: coral intent enters, violet intelligence interprets, cyan action progresses, teal evidence closes the loop.

## 5. Logo variants

- Primary horizontal wordmark: Convergence Core + `SYLVANDER` + optional `AGENT WORKSPACE` descriptor.
- App icon: mark centered on near-black field with generous safe area.
- Native compact: mark only.
- Terminal Unicode: `◆› SYLVANDER`.
- Terminal ASCII: `[o]> SYLVANDER`.
- Main agent marker: filled violet `◆`.
- Subagent marker: hollow violet `◇`.
- Agent team: `◆◇◇`.

## 6. Usage rules

- Do not put an `S` inside the native vector mark.
- Do not add eyes, a face, claws, or mascot animation.
- Do not use the full wordmark before every assistant response.
- Do not use gradients in terminal or small-size marks.
- Do not combine the mark with generic sparkle symbols.
- Preserve one mark-width of clear space around the native mark.
- At 16px, remove the inner ivory point before reducing stroke below one device pixel.
